This is a website on one page

This is a complete website on one page; the menu consists of tabs that open the linked content in the main section, which is where you are reading this. When a tab is clicked, the existing content in the main section disappears (hidden), and the content of the menu item is displayed. Everything is linked with an ID, and we use JavaScript to accomplish this. I believe it is now even possible to do this with ::before and ::after in CSS.

Some animations are used to make the content appear on the screen, which is done with CSS @keyframes, without any libraries.

A popup has also been added as an AI shop assistant; we can't do without it anymore, can we? Additionally, I consistently use only two columns in the main section; this is a very simple setup.
